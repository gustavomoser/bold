# Select

É um tipo de _input_ muito utilizado em formulários, onde o usuário está submetendo informações e precisa escolher apenas uma opção de uma lista. Quando o usuário insere algum texto no campo de input, as sugestões são filtradas. É importante apresentar opções de escolha quando ativo.

## Boas práticas

- A ordem dos itens da lista deve ser baseada na frequência de uso e, se aplicável, em ordem crescente.
- Caso seja necessário selecionar mais de uma opção, utilize o [Multi Select](/components/multi-select).
- Evite o uso do _select_ para _input_ de dados que são conhecidos pelo usuário, como datas de nascimento.

## Sugestão

Permita que o valor digitado seja o que o usuário digitar, fornecendo uma propriedade `createNewItem`.

Você pode usar a propriedade `openOnFocus` para controlar se o menu de sugestões deve aparecer quando a entrada receber o foco - o padrão é falso para as seleções de sugestões (quando createNewItem) e verdadeiro para as seleções normais.
