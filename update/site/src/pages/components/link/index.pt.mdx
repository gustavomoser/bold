# Link

Links são utilizados como elementos de navegação. Também podem mudar a forma de como o conteúdo é apresentado (ex.: ver mais, mostrar tudo).

## Boas práticas

- Utilize palavras ou frases curtas.
- Utilize labels descritivas que sejam consistentes com o destino do link. Evite labels como "Clique aqui", "Aqui".
- Se a ação resultar em troca ou alteração de dados, utilize o componente de [Botão](/components/button).
