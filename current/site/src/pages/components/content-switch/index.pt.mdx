# Content Switcher

O _Content Switch_ manipula o conteúdo apresentado na tela alternando entre as opções apresentadas através de botões, e por _default_ sempre tem uma opção selecionada. Utilize o _content switch_ em telas onde as opções são mutuamente exclusivas, permitindo que o usuário selecione apenas uma opção.

## Boas práticas

- Considere utilizar as boas práticas de [Botões](/components/button) para as labels.
- Sempre há uma opção selecionada por padrão.
