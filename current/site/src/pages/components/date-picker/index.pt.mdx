# Date picker

Componentes de data permitem que o usuário faça uma seleção específica através de um calendário. Por padrão, o calendário apresenta sempre o mês ou ano atual. Caso já tenha uma data inserida, o calendário apresenta o mês ou ano referente à esta data.

## Boas práticas

- O campo deve permitir que o usuário também insira o valor através de input do teclado.
- O campo deve ter apenas um tipo de entrada, e não deve ser divido em campos para cada informação (dia, mês, ano).

### Date Range Picker

- Utilize quando a data selecionada for próxima à data atual ou com um período menor que 1 ano.
- Utilize placeholder indicando o formato da entrada de dados, por exemplo: dd/mm/aaaa ou mm/dd/aa, etc.

### Week Range Picker

- Utilize-o caso os períodos selecionados devam começar em um domingo e terminar em um sábado.
- Caso coloque um valor na prop value do Week Range Picker, ele precisa ser uma semana, por exemplo: 02/05/2021 (Domingo) a 08/05/2021 (Sábado). Caso contrário, não vai marcar corretamente. Um exemplo de entrada inválida seria: 03/05/2021 (Segunda) a 05/05/2021 (Quarta).

### Month Range Picker

- Use quando o período selecionado deve ser no mínimo um mês inteiro.
- Se o usuário deve selecionar um período com maior granularidade, como semanas ou dias, utilize os componentes acima.
