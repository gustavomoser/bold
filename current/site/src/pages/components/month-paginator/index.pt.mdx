# Month Paginator

Permite ao usuário navegar através de meses e anos.

## Best practices

- Use o componente para controlar a navegação através de informações paginadas por mês.
- Se for necessário que o usuário selecione uma data específica, use o componente [Date Picker](/components/date-picker).
