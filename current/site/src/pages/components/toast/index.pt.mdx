# Toast

Utilize _toasts_ para mostrar informações que requerem mínima interação do usuário. As mensagens desaparecem após 5 segundos ou podem ser fechadas manualmente e são apresentadas nos cantos inferiores da tela. As mensagens não podem atrapalhar o fluxo de trabalho do usuário. São comumente utilizadas para alertar sobre _status_ de tarefas.

## Boas práticas

- Mensagens devem limitar seu corpo de texto em no máximo duas linhas.
- Devem desaparecer automaticamente após 5 segundos, e devem ter um ícone de fechar. Se entrar no _status_ de _hover_, a _toast_ deve ser mantida aberta.
- Podem ser incluídos links que redirecionam o usuário para próximos passos.
- Utilize ícones para transmitir clareza adicional. Os ícones informativos devem ser posicionados à esquerda do título.

**Este componente ainda não foi implementado como um componente em React.**
